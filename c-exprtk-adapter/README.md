# C exprtk adapter

Download exprtk the lib to user's include directory:

```shell
curl -L https://raw.githubusercontent.com/ArashPartow/exprtk/refs/heads/master/exprtk.hpp -o ${HOME}/.local/include/exprtk.hpp
```

## Build

### Debug

```shell
./debug.sh
```

### Release

```shell
./release.sh
```
